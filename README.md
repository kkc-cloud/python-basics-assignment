# python-basics-assignment
Python basic programs for assignment (grade checker, dictionary, file handling)





---

## **1. Grade Checker**

### Description
Program takes a score as input and prints the corresponding grade using if–elif–else conditions.

### File: `grade_checker.py`
Grading rules:
- 90+ → A  
- 80–89 → B  
- 70–79 → C  
- 60–69 → D  
- Below 60 → F  

---

## **2. Student Grades Manager**

### Description
A dictionary-based menu-driven program that allows the user to:
- Add a student and grade  
- Update an existing student’s grade  
- Print all students with grades  

### File: `student_grades.py`

Uses:
- Dictionary  
- While loop  
- If–else logic  

---

## **3. Write to a File**

### Description
Creates a text file and writes multiple lines of content using Python file handling:
- `open(filename, "w")`
- `write()`
- `with` block for auto-close  

### File: `write_file.py`

---

## **4. Read from a File**

### Description
Reads text from the file created earlier using:
- `open(filename, "r")`
- `read()`
- try–except for handling missing file  

### File: `read_file.py`
